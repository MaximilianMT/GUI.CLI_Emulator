Цель работы: выполнить дополнительные операции над графом зависимостей —
реализовать вывод порядка загрузки пакетов и сравнить его с эталонным менеджером пакето

Задачи этапа:
1) Поддержать режим вывода порядка загрузки зависимостей для заданного пакета.
2) Сравнить результаты работы программы с эталонным менеджером пакетов.
3) В случае расхождений — объяснить их наличие.
4) Продемонстрировать функциональность программы на тестовом репозитории.
5) Сохранить результат выполнения в репозиторий стандартным коммитом.

Реализация:
Файл программы: stage4.py
Язык: Python 3.13
Используемые библиотеки: argparse, sys

G: B


Команда для запуска: python3 stage4.py --repo test_graph.txt --package A --depth 3 --test
